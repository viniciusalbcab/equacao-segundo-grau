# Equação do Segundo Grau

Resolve equações do tipo **ax² + bx + c = 0** a partir dos coeficientes **a**, **b** e **c** informados pelo usuário.

## Como usar

```bash
python teste_cursor.py
```

Informe os coeficientes quando solicitado. O programa exibe o valor de **delta** (Δ = b² - 4ac) e as raízes conforme o caso:

- **Δ > 0:** duas raízes reais diferentes
- **Δ = 0:** duas raízes reais iguais (raiz dupla)
- **Δ < 0:** duas raízes complexas conjugadas

## Requisitos

- Python 3.x (biblioteca padrão)
